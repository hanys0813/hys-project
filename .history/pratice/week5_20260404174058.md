## 사용한 ai 프롬프트 ( 요약 )

    - 코드 구조 설명 요청

        AI에게 스네이크 코드 구조 설명을 해보았는데

        기본적인 로직은
        while 루프를 통해 게임이 실시간으로 구동됨
        방향키 입력을 받아 뱀의 좌표를 업데이트함
        if head == food 등의 좌표 비교로 충돌을 판정 함

    - 기획서 구체화 요청

        기획서 구체화 요청 과정에서

        게임 이름에 대해 집중을 하였습니다
        게임 이름을 일본 라노벨 느낌나게 지어달라 하였는데
        평범한 뱀이었던 내가 알고 보니 공주를 구할 기사? ~1000점 모아 보스를 물리치고 해피엔딩을 맞이하겠어~ 제목이 나오게 되었습니다.

        노동에 가까웠다면, 본 게임은 특정 점수대마다 새로운 능력이 해금되는 성장형 시스템을 채택하였습니다

    - 구현 가능성 확인

        구현 가능성에 대해 질문을 해본 결과
        ai는 구현이 충분히 된다고 하였습니다.

## ai 답변에서 도움이 된것

        난이도 설정 하는게 굉장히 도움이 된 것 같습니다.
        난이도 밸런스 맞추는게 굉장히 어려운데 
        난이도 밸런스 맞추는 부분에서 도움이 된것 같습니다
        ai가 난이도를 4단계로 분류해서 만들어주었습니다
